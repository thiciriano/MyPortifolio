# Thiago Neves — Frontend Developer
Minimalist personal portfolio built with **pure HTML, Tailwind CSS (CDN), jQuery**, and a strong focus on **accessibility**, **performance**, and **clean design**.

Live demo → [thiciriano.github.io](https://thiciriano.github.io/MyPortfolio) *(or your GitHub Pages link)*

## Features

- **Single-file** project (no build tools, no dependencies beyond CDNs)  
- Fully **responsive** (mobile-first)  
- **Dark / Light mode** with system preference detection and persistence  
- **100% accessible** (semantic HTML, proper ARIA, keyboard navigation)  
- Smooth hover effects and grayscale-to-color image transitions  
- Uses **Lucide Icons** for lightweight, beautiful SVG icons  
- Fast loading — no frameworks, no bloat
